Database Security: Concepts, Approaches, and Challenges

This book provides a comprehensive overview of database security concepts, including data confidentiality, integrity, and availability. It introduces foundational security models, discusses privacy-preserving techniques, and addresses the challenges faced by modern database systems in the era of cloud computing and big data. It's an excellent resource for understanding both theoretical and practical aspects of database security.

Authors: Sushil Jajodia, Pierangela Samarati, and Vijayalakshmi Atluri

Pages: 427

Rating: 4.1/5 (4210)


Database and Applications Security: Integrating Information Security and Data Management

This book integrates database security with broader information security practices, offering a detailed examination of how to secure databases and applications in tandem. It covers a wide range of topics, including access control, inference control, auditing, and the protection of both data and metadata. It’s particularly valuable for those looking to understand how security can be embedded across data management systems and their applications.

Author: Bhavani Thuraisingham

Pages: 902

Rating: 4.6/5 (931)


Data Security and Privacy in Databases: Trends and Challenges

This book examines recent trends and emerging challenges in database security, with a focus on privacy protection. It covers privacy-preserving data mining, anonymization techniques, and compliance with data protection regulations like GDPR. It also discusses securing big data environments and distributed databases. Ideal for readers interested in both traditional and cutting-edge database security topics.

Authors: Alfredo Cuzzocrea, Mohamed Gaber, and Srinivas Sampalli

Pages: 657

Rating: 4.3/5 (4207)

This lecture discusses the role of encryption in protecting data at rest and in transit within database systems. It covers encryption techniques, key management, and access control mechanisms like role-based and attribute-based access control, helping secure data from unauthorized users.
